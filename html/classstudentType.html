Detailed Description

Represents a student with GPA, classification and ID.

@invariants

  • gpa is maintained in the range [0.0, 4.0].
  • id and classification are valid std::string instances and may hold sentinel values such as "NOT SET" but are never left indeterminate.
  • The personType invariants apply for the inherited fields.
